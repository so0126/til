# TIL GitHub Pages 구현 계획표

> **콘셉트:** Blue Summer Study Minihome — 파란 바다 근처 작은 미니홈피 공부방
> **원칙:** Markdown 가독성 > 탐색 편의 > 검색 > 미니홈피 감성 > 캐릭터/장식

> [!IMPORTANT] 핵심 설계 원칙
> **SaaS 대시보드나 일반적인 개발 블로그처럼 만들지 말고, 화면 중앙에 독립된 미니홈피 프레임이 있는 개인 다이어리형 레이아웃으로 구현해라. 캐릭터와 바다 장식은 보조 요소이며, 핵심은 아코디언 탐색과 Markdown 가독성이다.**
>
> 피해야 할 것: 흰색 개발 블로그 템플릿 / SaaS 대시보드 / 보라 그라데이션 AI 서비스 / 네온 사이버펑크 / 분홍 귀여운 디자인 / 캐릭터가 화면 대부분 차지 / 의미없는 그래프와 통계

## 🎨 디자인 스펙 체크리스트

### 컬러 시스템
- ✅ Ocean 색상 변수 (`--ocean-900` ~ `--ocean-100`)
- ✅ 배경 그라데이션 (`#DFF3FA`)
- ✅ 카드 아이보리 (`#FFFDF7`)
- ✅ 포인트 컬러 (yellow, pink, green, coral)

### 폰트
- ✅ 통일 폰트: SUIT (본문, 제목, UI)
- ✅ 코드 폰트: JetBrains Mono / D2Coding

### 레이아웃
- ✅ 중앙 미니홈피 프레임 (max-width 1280px)
- ✅ 사이드바 260px (명세는 280px)
- ✅ 3열 레이아웃 (사이드바 + 본문 + TOC)
- ✅ 둥근 모서리 / 옅은 테두리 / 부드러운 그림자

### 헤더
- ✅ 사이트명 + 짧은 문구
- ✅ TOTAL (글 수) | TODAY (방문) | GitHub | Random
- ✅ 다크모드 버튼
- ❌ 최근 업데이트 날짜

### 사이드바
- ✅ 프로필 카드 (GitHub 이미지, 이름, 소개)
- ✅ 최근 업데이트 날짜
- ✅ 아코디언 (다중 열기, 연도/월, localStorage 유지)
- ✅ 글 개수 배지
- ✅ 현재 글 강조 (active 스타일)
- ✅ 현재 글 ⭐ 아이콘 표시 + aria-current="page"
- ✅ 사이드바 독립 스크롤
- ❌ 캐릭터 상태 메시지

### 홈 페이지
- ❌ README INTRO 파싱 + 카드 렌더링
- ✅ 히어로 캐릭터 문구 `( ˶ˆᗜˆ˵ ) ✨ 열공 중이에요!`
- ✅ 검색창 (즉시 갱신, 초기화 버튼)
- ✅ 날짜 퀵버튼
- ✅ 카테고리 다중선택
- ✅ 활성 필터 배지 (× 삭제)
- ✅ 글 카드 (날짜, 카테고리, 제목, 미리보기, 읽기 시간)
- ❌ 글 카드 태그 표시
- ✅ 통계 섹션 (조개형 활동, 연속 작성일, 전체/이번달)
- ✅ 홈 페이지네이션

### 상세 페이지
- ✅ 헤더 (카테고리, 제목, 날짜, 읽기 시간, 태그, GitHub 링크)
- ✅ Markdown 렌더링 (GFM)
- ✅ 코드블록 복사 버튼
- ✅ 코드블록 언어명 표시 + 창 버튼 3개 (macOS 스타일)
- ✅ 표 (파란 헤더, hover)
- ✅ GitHub 경고 박스 (Note/Tip/Warning 등)
- ✅ TOC (자동생성, 스크롤 연동, 맨 위로 버튼)
- ✅ 이전/다음 글 카드

### 장식/분위기
- ✅ 배경 구름 (CSS 이모지)
- ✅ 배경 물결선 (CSS)
- ❌ 실제 장식 이미지 (우표, 마스킹테이프, 폴라로이드 등)
- ✅ 귀여운 캐릭터 텍스트 `( ˶ˆᗜˆ˵ ) ✨`

### 빈 화면/오류
- ✅ 404 페이지 (캐릭터, 메인으로 버튼)
- ✅ 검색 결과 없음 메시지

### 접근성
- ✅ aria-label (버튼, 링크 전반)
- ✅ aria-current="page" (활성 글 링크)
- ✅ aria-expanded (아코디언 버튼)
- ✅ prefers-reduced-motion 지원
- ❌ 키보드 아코디언 조작 (Tab+Enter는 기본 지원, 방향키 미지원)
