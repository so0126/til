2026-06-22 프로그래머스 12390 문자열 뒤집기 (Lv.0) example

  • 자료구조/알고리즘: [[자료구조_문자열]], [[자료구조_배열]]
  • 핵심 메서드: split → reverse → join 패턴
  • 상태: #성공

💻 내 풀이

function solution(my_string) {
    return my_string.split('').reverse().join('');
}

💡 다른 사람의 풀이 보고 배운 점

스프레드 연산자로 더 짧게 쓸 수 있음

function solution(my_string) {
    return [...my_string].reverse().join('');
}

⚠️ 자주 하는 실수 / 기억할 것

  • reverse()는 배열에만 쓸 수 있음 → 문자열은 반드시 split('') 후 사용
  • split('')과 [...str]은 동일 결과 → 스프레드가 더 모던한 스타일
