# 🧩 자료구조: 큐 (Queue)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 개념 — FIFO (First In First Out)
```js
const queue = [];
queue.push(1);   // 삽입 (뒤)
queue.shift();   // 꺼내기 (앞) ← O(n) 주의!
```

**쓰이는 곳:** BFS, 프린터 대기열, 순서 처리

## 성능 주의
JS에서 `shift()`는 O(n) → 큰 입력에선 인덱스 포인터 방식 사용
```js
let head = 0;
while (head < queue.length) {
  const cur = queue[head++];
}
```
