# 🧩 알고리즘: 구현 / 시뮬레이션

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 유형 특징
- 문제에서 시키는 대로 그대로 구현
- 조건이 많고 복잡하지만 알고리즘 자체는 단순
- 방향 이동, 좌표 계산, 상태 전이가 자주 나옴

## 자주 쓰는 패턴
```js
// 4방향 이동 (상하좌우)
const dr = [-1, 1, 0, 0];
const dc = [0, 0, -1, 1];

for (let d = 0; d < 4; d++) {
  const nr = r + dr[d];
  const nc = c + dc[d];
  if (nr < 0 || nr >= N || nc < 0 || nc >= M) continue;
  // 로직 처리
}
```
