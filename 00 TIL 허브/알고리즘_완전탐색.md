## BFS (너비 우선 탐색) — 최단거리
```js
function bfs(start) {
  const queue = [start];
  const visited = new Set([start]);
  while (queue.length) {
    const cur = queue.shift();
    for (const next of graph[cur]) {
      if (!visited.has(next)) {
        visited.add(next);
        queue.push(next);
      }
    }
  }
}
```

## DFS (깊이 우선 탐색) — 경우의 수
```js
function dfs(node, visited) {
  visited.add(node);
  for (const next of graph[node]) {
    if (!visited.has(next)) dfs(next, visited);
  }
}
```

- 순열/조합 탐색
