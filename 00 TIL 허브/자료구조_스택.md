# 🧩 자료구조: 스택 (Stack)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 개념 — LIFO (Last In First Out)
```js
const stack = [];
stack.push(1);          // 삽입
stack.pop();            // 꺼내기 (뒤에서)
stack[stack.length-1];  // peek
```

**쓰이는 곳:** 괄호 짝 맞추기, DFS, 뒤로 가기 기능
