# 🧩 알고리즘: 정렬 (Sort)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 시간복잡도 비교
| 알고리즘 | 평균 | 최악 | 안정성 |
|---------|------|------|--------|
| 버블 | O(n²) | O(n²) | 안정 |
| 선택 | O(n²) | O(n²) | 불안정 |
| 삽입 | O(n²) | O(n²) | 안정 |
| 병합 | O(n log n) | O(n log n) | 안정 |
| 퀵 | O(n log n) | O(n²) | 불안정 |
| 힙 | O(n log n) | O(n log n) | 불안정 |

## JS 정렬 패턴
```js
arr.sort((a, b) => a - b);          // 오름차순
arr.sort((a, b) => b - a);          // 내림차순
arr.sort((a, b) => a.localeCompare(b)); // 문자열 사전순
arr.sort((a, b) => a[1] - b[1]);    // 2번째 원소 기준
```
