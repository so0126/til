# 🧩 자료구조: 해시 (Hash / Map)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 언제 쓰나?
- 빈도 카운팅 (단어 등장 횟수)
- 중복 체크
- 두 배열의 교집합/합집합
- O(1) 빠른 조회가 필요할 때

## Map vs Object
| | Map | Object |
|--|-----|--------|
| 키 타입 | 모든 타입 | String/Symbol |
| 순서 보장 | O (삽입 순) | X |
| 크기 | `.size` | `Object.keys().length` |

## 핵심 메서드 (Map)
```js
const map = new Map();
map.set(key, value);
map.get(key);
map.has(key);
map.delete(key);
map.size;
```
