# 🧩 자료구조: 문자열 (String)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 핵심 메서드 (JavaScript)
| 메서드 | 설명 |
|--------|------|
| `split(sep)` | 문자열 → 배열 |
| `join(sep)` | 배열 → 문자열 |
| `replace(a, b)` | 첫 번째만 교체 |
| `replaceAll(a, b)` | 전체 교체 |
| `includes(str)` | 포함 여부 |
| `indexOf(str)` | 위치 반환 |
| `slice(s, e)` | 부분 문자열 |
| `trim()` | 양쪽 공백 제거 |
| `toLowerCase()` | 소문자 변환 |
| `repeat(n)` | n번 반복 |

## 자주 쓰는 패턴
```js
// 문자열 뒤집기
str.split('').reverse().join('')

// 문자 빈도 세기
[...str].reduce((acc, c) => (acc[c] = (acc[c]||0)+1, acc), {})
```
