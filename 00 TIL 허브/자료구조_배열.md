# 🧩 자료구조: 배열 (Array)

> 그래프 뷰 허브 노트 — 이 개념을 사용한 코테 풀이들이 위성처럼 연결됩니다.

## 핵심 메서드 (JavaScript)
| 메서드 | 설명 |
|--------|------|
| `slice(s, e)` | 원본 유지, 일부 추출 |
| `splice(s, n)` | 원본 수정, 삭제/삽입 |
| `indexOf(v)` | 첫 번째 위치 반환 |
| `includes(v)` | 포함 여부 Boolean |
| `map(fn)` | 변환 후 새 배열 |
| `filter(fn)` | 조건 통과 원소만 |
| `reduce(fn, init)` | 누적 계산 |
| `sort((a,b)=>a-b)` | 오름차순 정렬 |
| `flat(depth)` | 중첩 배열 펼치기 |
