    ❌ 틀린 문제 & 깨달은 점

    문제 통째로 적지 말고, '내가 몰랐던 개념/핵심'만 한 줄로 요약

    1. 지연 갱신 VS 즉각 갱신

      • 헷갈린 점: 용어가 헷갈림 기억이 안 남
      • 한줄 정리: 지연 갱신은 완료될 때 까지 기다림, 즉각 갱신은 바로 개입함
      • 연결 개념: 로그 기반 회복 - 데이터베이스 변경 사항 로그 파일에 순차 기록,
        장애 발생 시 복구

    2. 그림자 페이징

      • 헷갈린 점: 수정 시 원본(그림자) 페이지 안 건드리고 새 페이지에 기록한다. 현재 페이지 테이블만 새 페이지를 가리킨다.
      • 한줄 정리: 그림자페이지는 원본을 꼬옥 갖고있어주면 돼~ 든든한 백업
      • 연결 개념: 페이지 - 데이터 묶음
        페이지 테이블 - 주소록(몇 번 페이지 디스크 어디에 저장되있는지 적어둔 표)

    3. SQL문

      • 헷갈린 점: 그냥 모든게 다 헷갈림 처음부터 다시 읽으면서 문제 풀어야 함
      • 한줄 정리: DDL 정의어 ALTER 테이블 DML 조작어 SELECT 데이터 DCL 제어어 GRANT 권한
      • 연결 개념: CONSTRAINT FOREIGN KEY REFERENCES

    4. 데이터 정의어 조작어 제어어 헷갈림

      • 헷갈린 점: DELETE나 DROP이나 ALTER나... 뭐랄까 뭐가 뭔지 모르겠달까
      • 한줄 정리: DDL 정의어 ALTER 테이블 DML 조작어 SELECT 데이터 DCL 제어어 GRANT 권한
      • 연결 개념: CASCADE / RESTRICT -> 연쇄 회수/회수 거부(옵션 생략 시 기본)
