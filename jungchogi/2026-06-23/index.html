❌ 틀린 문제 & 깨달은 점

  1. 디비젼(A÷B)

    • 헷갈린 점: B 릴레이션의 모든 값과 짝을 이루는 A의 값만 골라내라.
    • 한줄 정리: 모든 값과 짝을 이뤄야 함. 그 중 몇 개만 만족 X
    • 연결 개념: #관계대수연산자

  2. 데이터 모델 3가지 구성요소

    • 헷갈린 점: 구조, 연산, 제약조건 개념 자체
    • 한줄 정리: 구조는 어떻게 나타낼 것인지 표현, 연산은 데이터 처리 작업 명세, 제약조건은 데이터 무결성 유지를 위한 제약
    • 연결 개념: #데이터모델

  3. 교집합∩과 조인**⋈**

    • 헷갈린 점: 개념 차이
    • 한줄 정리: 교집합은 위아래로 합침(공통된 행 찾아내기), 조인은 옆으로 붙임(새로운 정보 만들기)
    • 연결 개념: 합집합 u, 차집합 -, 카티션 프로덕트 x (행렬) #관계대수연산자

  4. 정규형

    • 헷갈린 점: 개념
    • 한줄 정리: 1) 하나셀 단일값만 2) 기본키 2개 이상일 때 3) 기본키 아닌 놈들끼리 꼬리물기 분리해서 제거 4) BCNF 결정자(식별자)인데 후보키(유일, 최소)가 아니라면 쪼개기
    • 연결 개념: 유일성(중복 금지), 최소성(군더더기 금지) #정규화

  5. 해싱 접근

  • 헷갈린 점: 해싱 접근, 해시 충돌, 해시 주소, 해시 함수 -> 용어
  • 한줄 정리: "해싱 접근"외 다 해시
  • 연결 개념: 레드 블랙 트리 체이닝(Java) #레드블랙트리
  1. 관계 대수 vs 관계 해석
  • 헷갈린 점: 관점따라 도메인의 뜻이 다름, 관계 대수 - 어떻게 계산할까? How 관계 해석 - 어떤 데이터? What
  • 한줄 정리: 실무-이름표: 이 데이터는 무슨 정보인가? 수학-울타리: 이 칸에 어떤 값만 들어올 수 있나? 관계 해석-빈칸(x,y): 도메인 변수 조건식 짤 때 이 칸의 값을 뭐라고 부를까?
  • 연결 개념: 절차적 언어(관계 대수), 비절차적 언어(관계 해석, SQL)
